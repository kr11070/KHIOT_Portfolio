뉴스 쉬운말 모드

네이버뉴스 · 다음뉴스 · 한국경제 기사 페이지에서 플로팅 버튼 하나로 본문을 쉬운 말로 바꿔주는 Chrome 확장 프로그램입니다.

  • 지원 사이트 진입 시 좌측 하단에 "쉬운말" 버튼 자동 표시
  • 클릭하면 AI(Groq)가 본문을 쉬운 문장으로 재구성
  • 다시 클릭하면 원문으로 즉시 복원

이 확장 프로그램은 Chrome Web Store가 아닌 "개발자 모드"로 직접 로드하는 방식입니다. 소스 코드와 설치 방법은 저장소의 public/projects/news-reading-mode/README.md를 참고하세요.
