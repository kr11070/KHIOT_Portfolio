# 뉴스 쉬운말 모드 — 기획문서

## 배경

Agile Squad 프로젝트에서 다룬 "읽기 난이도 전환" 아이디어를, 특정 서비스 하나가 아니라 네이버뉴스·다음뉴스 등
주요 뉴스 사이트 어디서든 쓸 수 있는 브라우저 확장 프로그램으로 확장했습니다.

## 목표

뉴스 기사를 읽다가 어렵게 느껴질 때, 버튼 하나로 바로 쉬운 말로 바꿔 볼 수 있게 한다.

## 주요 기능

- 지원 사이트(네이버뉴스, 다음뉴스, 한국경제, 중앙일보, 동아일보, 한겨레, 연합뉴스, 조선일보) 기사 페이지 진입 시 플로팅 버튼 자동 표시
- 버튼 클릭 시 AI(Groq API)가 기사 본문을 쉬운 문장으로 재구성
- 다시 클릭하면 원문으로 즉시 복원 (재변환 없이 캐시된 결과 재사용)

## 구조

- **확장 프로그램(`extension/`)**: 기사 본문 추출 + 플로팅 버튼 UI + 백엔드 호출
- **백엔드(`server/`)**: Netlify Function에서 Groq API 키를 안전하게 보관하고 텍스트 변환 요청을 대신 처리

## 향후 개선 아이디어

- 원문/쉬운말 외에 "쉽게읽기" 단계 추가 (Agile Squad 프로토타입처럼 3단계 슬라이더)
- 조선일보 선택자 실제 검증 및 지원 사이트 추가 확장
- 변환 결과 캐싱(같은 기사 재방문 시 재요청 없이 즉시 표시)
