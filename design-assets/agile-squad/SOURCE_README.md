# 한경 UXUI 5기 — Agile Squad 팀 프로젝트

> **사용자 리서치와 과업 플로우 설계를 기반으로 한 금융 서비스 분야의 한국경제신문 도메인 신규 앱 인터랙티브 프로토타입 제작**

---

## 프로젝트 개요

| 항목 | 내용 |
|------|------|
| **과정** | 프로젝트 기반 UXUI 디자인 실전 캠프 5기 |
| **팀명** | Agile Squad |
| **기간** | 2026년 6월 19일 ~ 7월 3일 (총 80시간) |
| **방법론** | Agile Scrum · Google Design Sprint · UX Research |
| **협업 툴** | Jira · GitHub · Figma / FigJam |

---

## 프로젝트 목표

한국경제(한경) 서비스의 현황을 분석하고, **2030 신규 타겟 사용자**를 대상으로 한 금융 서비스 도메인의 신규 앱 UX/UI를 기획·설계한다.

1. 한국경제 서비스 현황 분석 → 개선 필요 사항 정의
2. 사용자 리서치(가설 수립 → 인터뷰/게릴라 테스트) → 정성·정량 데이터 확보
3. 과업 플로우(User Flow) · IA · 와이어프레임 설계
4. Figma 인터랙티브 프로토타입(MVP) 제작 및 발표
